# Design: feature-industry-catalyst-analysis

## Context
- 本设计承接 `pre_design.md`：行业/催化分析需要成为长期知识库节点，并由个股日评引用与增量更新。
- 现有 `vault/companies/` 与 `vault/stock/<date>/` 工作流继续保留，不迁移到全新体系。
- 项目尚未开始真实使用，因此允许对已有入口做安全追加，但仍不得覆盖用户已有正文。
- 规格已定义两个新增能力：`industry-catalyst-knowledge` 与 `peer-catalyst-comparison`。
- 规格同时修改 `obsidian-knowledge-base` 与 `obsidian-report-scaffolding`，使长期入口和日评模板接入行业/催化节点。

## Goals / Non-Goals
- Goals：本次设计要落地行业/催化长期节点、横向比较框架、日评引用与增量沉淀，并保持现有 vault 工作流兼容；详见 `pre_design.md` §1。
- Non-Goals：不做自动行业分类、概念归属、同催化股票池生成、量化评分、因子模型、排名模型或具体投资价位建议；详见 `pre_design.md` §1。

## Decisions
- 节点目录：行业节点放在 `vault/knowledge/industries/`，催化节点放在 `vault/knowledge/catalysts/`，文件名使用可读中文主题名；目录只表达用户人工选择的节点类型，不做自动分类判断。
- 节点模板：行业/催化节点共享核心章节骨架，包含 YAML frontmatter、tags、一级标题、核心结论、相关股票、关键催化或行业逻辑、风险因素、更新记录和来源清单；tags 区分 `stock/industry` 与 `stock/catalyst`。
- 入口接入：长期知识库入口新增 `[[行业研究入口]]` 与 `[[催化研究入口]]`，两个独立入口页负责说明、索引骨架和后续整理，总入口不直接维护所有节点列表。
- 日评分工：个股日评新增独立 `行业/催化横向比较` 章节，记录当日比较口径、关联节点、对比对象、共同驱动、相对差异、共同风险和证据链接；行业/催化长期节点记录可复用结论与更新记录。
- 章节位置：日评主体顺序采用 `行业与政策` → `行业/催化横向比较` → `竞争格局`，使横向比较承接行业政策判断并导向竞争格局分析。
- 幂等策略：初始化脚本允许对已有入口页查重后追加缺失的行业/催化入口链接或入口区块，但不得覆盖整文件；找不到稳定插入点时跳过并在摘要中提示。
- 证据链分工：日评保留当日分析证据；行业/催化节点保留可复用证据索引，并链接相关日评或证据库入口；两边都保留来源、发布日期或检索时间、数据日期字段。
- 迁移策略：本 change 作为早期模板升级处理；新初始化生成完整行业/催化结构，已有长期知识库入口安全补链接，已有日评和公司档案不批量改写。

```text
vault/
  knowledge/
    行业研究入口.md
    催化研究入口.md
    industries/
      <行业主题>.md
    catalysts/
      <催化主题>.md
  companies/
    <code>-<name>.md
  stock/<date>/
    <code>-<name>-日评.md  --> [[<行业主题>]] / [[<催化主题>]]
```

## Risks / Trade-offs
- [Risk] 安全追加入口可能找不到稳定插入点 → [Mitigation] 跳过追加并在初始化摘要中提示用户手动补链接。
- [Risk] 行业与催化主题边界可能重叠 → [Mitigation] 由用户人工选择目录和节点类型，不做自动分类或概念归属判断。
- [Risk] 日评与长期节点可能重复或漏同步 → [Mitigation] 日评保留当日证据，并在沉淀提示中列出建议写入行业/催化节点的信息。
- [Risk] 新章节改变日评模板结构 → [Mitigation] specs 和 tests 关注必备章节与字段，不依赖旧的固定 14 节数量。
- [Risk] 不做自动股票池或评分模型会限制自动化程度 → [Mitigation] 本次只保证框架与沉淀能力，后续可作为独立 change 扩展。

## Migration Plan
- 更新模板，使新生成的长期知识库入口、行业入口、催化入口、行业/催化节点和个股日评包含完整链接与章节骨架。
- 更新初始化脚本，使新 vault 或缺失文件按完整结构生成。
- 对已有长期知识库入口执行查重安全追加，补充 `[[行业研究入口]]` 与 `[[催化研究入口]]` 或对应入口区块。
- 不批量改写已有个股日评、公司档案或用户已编辑的行业/催化节点。
- 初始化摘要必须标记创建、跳过和无法安全追加的文件或链接。
